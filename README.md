
# Language Translator

A Python-based language translator that translates text between English, Hindi, and Telugu using the `deep-translator` library.

## Features

- Translate text between multiple languages
- Automatic language translation
- View translation history
- Clear translation history
- Menu-driven command-line interface
- Handles invalid user input

## Technologies Used

- Python
- deep-translator
- File Handling

## Project Structure

```
language-translator-python/

```

## Installation

Clone the repository:

```bash
git clone https://github.com/lalithreddy2304/language-translator-python.git
```

Install the required library:

```bash
pip install deep-translator
```

Run the program:

```bash
python language_translator.py
```

## How It Works

1. Select **Translate Text** from the menu.
2. Choose the source language.
3. Choose the target language.
4. Enter the text to translate.
5. View or clear translation history whenever needed.
